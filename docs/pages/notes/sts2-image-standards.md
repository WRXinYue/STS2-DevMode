Reference pixel dimensions for character select, energy, cards, powers, relics, potions, and mod cover art. **Body in Chinese.**

:::

::: zh-CN

## 角色选择界面

| 用途                        | 尺寸                                        |
| --------------------------- | ------------------------------------------- |
| 选角按钮头图（普通 / 锁定） | 132 × 195                                   |
| 角色图标                    | 128 × 128                                   |
| 地图标记                    | 128 × 128                                   |
| 角色立绘（战斗场景静态图）  | 参考原版 1448 × 1520，当前占位图 200 × 400  |
| 联机玩家头像                | 64 × 64                                     |

## 能量

| 用途       | 尺寸    |
| ---------- | ------- |
| 能量大图标 | 74 × 74 |
| 能量文字图 | 24 × 24 |

## 卡牌

| 用途                 | 尺寸       |
| -------------------- | ---------- |
| 卡牌立绘（游戏内）   | 250 × 190  |
| 卡牌立绘（放大预览） | 1000 × 760 |

## 能力（Powers）

| 用途                 | 尺寸      |
| -------------------- | --------- |
| 图标（小）           | 64 × 64   |
| 图标（大，悬停显示） | 256 × 256 |

## 遗物（Relics）

| 用途                 | 尺寸      |
| -------------------- | --------- |
| 图标（小）           | 94 × 94   |
| 轮廓图               | 94 × 94   |
| 图标（大，悬停显示） | 256 × 256 |

## 药水（Potions）

| 用途 | 尺寸      |
| ---- | --------- |
| 图标 | 256 × 256 |

## Mod 封面

| 用途   | 尺寸      |
| ------ | --------- |
| 封面图 | 420 × 420 |
